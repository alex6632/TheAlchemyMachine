# The Alchemy Machine
The Alchemy Machine is a potion simulator

## Requirements

* node.js v8.9 or later
* angular v^6.0.0 or later

## Install

On first install, run
```
npm run setup
```

## Run

### Spectator mode
Run 
```
npm run start
```

### Developer mode
1/ Run node server
```
npm run start-api
```
or just
```
nodemon
```

2/ Run angular
```
npm run start-client
```

## Front

Just go to http://localhost:4200
