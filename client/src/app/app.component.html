The Alchemy Machine

Voici un "simulateur" de création de potions alchimiques réalisé avec Node.js et Angular

L'objectif de ce jeu est de trouver les 10 potions magiques. Si vous ne trouvez pas les bons mélanges, les solutions sont disponibles en cliquant sur le bouton Aide.

  • Diamant + Hélium liquide + Emeraude
  • Nautile + Requin grande gueule + Vinaigre
  • Sang + Cendres + Eau régale
